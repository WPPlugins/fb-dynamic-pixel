== Description ==

Sometimes you think about the Facebook pixel or Facebook retargeting pixel and how is it working? The answer is here! <strong>FB Dynamic Pixel</strong> plugin for WordPress and WooCommerce helps you very well when you are planning to create very successful Facebook campaigns, very relevant audiences and high level conversions.

* You will be able to install <strong>Facebook Dynamic Events</strong> for <strong>AddToCart</strong> and <strong>AddToWishList</strong> Events, You will able to install some events for some posts, pages or products.
* This plugin lets you choose what are exact information you want to send to Facebook pixel! All Facebook categories/events are available to install with this plugin.
* The <strong>Search</strong> Event and <strong>CompleteRegistration</strong> Event will be installed with just one click.
* This plugin adds a new meta box for WooCommerce products add/edit page, You will be able to install one event once the product is posted if you want this. When you need more than event for the same product you will be able to install any number of events for this product from the “Manage Event” page.
* It is not just WordPress Facebook Pixel Plugin to install the Facebook pixel, This plugin already helps you to make your remarketing or retargeting campaigns more effective and get more success.

== Free Features ==

* Plugin requires only your Facebook Pixel ID.
* Install default event (PageView) with the default pixel code.
* You have ability to choose the places which will pixel and events should be installed.
* You can install AddToCart for Woocommerce by default to run when the visitor click on the button.
* Search event installed automatically by the “search_string” parameter, and you can prevent this event also.
* Professional WooCommerce Integration.
* You have ability to install new events based on Facebook Standard Events for every post, page or Woocommerce product manually in easy way.
* You have ability to install custom events not only the Facebook Standard Event.
* You will able to add any custom parameters to any events you may be install.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `FB Dynamic Pixel` from Plugins page.

The plugin will add a new menu called `FB Dynamic Pixel`. Goforward using the plugin from here!
